Program Kerja Lembaga dan Departemen

Ketua

- Progam penyusuna Progam kerja untuk seluruh divisi

- Program Kepemimpinan: Workshop dan pelatihan untuk meningkatkan keterampilan kepemimpinan.

- Program Perwakilan: Menghadiri dan mewakili organisasi di acara resmi.

- Program Evaluasi: Melakukan evaluasi rutin terhadap kinerja organisasi.

Sekretaris

- Program Administrasi: Meningkatkan efisiensi administrasi melalui digitalisasi dokumen.

- Program Pengarsipan: Menyusun sistem pengarsipan yang terorganisir.

- Program Notulensi: Pelatihan pembuatan notulen rapat yang efektif.

Bendahara

- Program Manajemen Keuangan: Pelatihan pengelolaan keuangan yang efisien.

- Program Audit Internal: Melakukan audit keuangan secara berkala.

- Program Penggalangan Dana: Mengadakan kegiatan untuk mengumpulkan dana organisasi.

Departemen Organisasi

- Program Pengembangan Struktur Organisasi: Mengkaji dan memperbaiki struktur organisasi.

- Program Penyusunan SOP: Menyusun dan menyosialisasikan Standar Operasional Prosedur.

- Program Evaluasi Kinerja Organisasi: Mengadakan evaluasi kinerja secara berkala.

- Program Pengembangan Kapasitas Anggota: Mengadakan pelatihan dan workshop untuk anggota.

- Program Komunikasi dan Informasi: Membuat sistem komunikasi internal yang efektif.

- Program Penyusunan Rencana Strategis: Menyusun dan meninjau rencana strategis organisasi.

Departemen Kaderisasi

- Program Pelatihan Kepemimpinan: Mengadakan pelatihan kepemimpinan untuk anggota baru.

- Program Rekrutmen Anggota: Kampanye untuk menarik anggota baru ke dalam organisasi.

- Program Pembinaan Anggota: Memberikan bimbingan dan pelatihan kepada anggota yang sudah ada.

Departemen Dakwah

- Program Ceramah dan Pengajian: Mengadakan ceramah dan pengajian rutin untuk anggota.

- Program Kampanye Nilai-Nilai Agama: Menyebarkan ajaran dan nilai-nilai agama melalui berbagai media.

- Program Pembinaan Rohani: Memberikan bimbingan rohani kepada anggota.

Departemen Olahraga, Seni, dan Budaya

- Program Kompetisi Olahraga: Mengadakan kompetisi olahraga antar anggota.

- Program Pameran Seni: Mengadakan pameran seni untuk menampilkan karya anggota.

- Program Pelestarian Budaya: Mengadakan kegiatan untuk melestarikan budaya lokal.
